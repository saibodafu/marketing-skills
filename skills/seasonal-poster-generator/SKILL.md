# 节气营销海报生成

本技能用于把「节气 + 产品」转化为可直接执行的品牌节气海报方案。

核心目标不是单独写节气祝福，也不是泛泛输出文案，而是完成一条完整创意链路：

```text
节气意象 -> 产品价值 -> 海报主张 -> 视觉场景 -> AI 生图提示词 -> 成品海报
```

## 使用场景

适用于以下任务：

- 二十四节气品牌海报
- 小红书、朋友圈、公众号头图节气视觉
- 电商活动图、产品上新图、节气借势图
- 品牌社交媒体节气内容
- 需要「文案 + 视觉创意 + 生图提示词 + 成品图」的一站式海报生成

## 触发条件

当用户出现以下表达时，应优先使用本技能：

- “立夏，宠物猫粮”
- “帮我做一张节气海报”
- “节气 + 产品，直接出图”
- “生成节气海报提示词”
- “给这个产品做二十四节气借势海报”
- “输出节气文案和生图 prompt”
- “做成一张可用的节气品牌海报”

如果用户只要求产品买点短文案，应优先使用 `product-copywriting`。如果用户只要求情绪短句，不需要节气视觉和生图提示词，可考虑 `jiangxiaobai-style-copywriting`。

## 必要输入

用户至少需要提供：

- 节气名称
- 产品名称或产品品类

可选输入：

- 产品卖点
- 品牌名称
- 品牌调性
- 目标人群
- 海报用途
- 画面风格
- 尺寸比例
- 是否需要真实品牌包装露出

如果用户只提供「节气 + 产品」，不要追问，直接基于品类常识做轻量假设，并在输出中标注“假设”。

只有当缺少节气或产品品类，导致无法判断主题时，才追问：

> 你想做哪个节气、哪个产品或品类的海报？

## 默认处理方式

信息不足但不影响生成时，按以下默认值处理：

- 品牌调性未知：默认“自然、克制、东方节气美学、现代商业海报质感”。
- 产品卖点未知：按品类常见卖点做轻量假设，但不得编造数据、认证、销量、功效或真实案例。
- 目标人群未知：按该品类最常见购买者推断。
- 用途未知：默认适合小红书、朋友圈和品牌社交媒体。
- 尺寸未知：默认竖版 4:5。
- 品牌名未知：使用“无具体品牌名的产品包装”，不得虚构真实品牌 Logo。

必须把关键假设写出来，不要伪装成用户已提供的信息。

## 核心规则

1. 不输出陈词滥调、空泛祝福或模板化节气口号。
2. 避免高频俗套表达，例如“愿你”“不负时光”“美好如约而至”“把日子过成诗”“万物可爱”“人间值得”。
3. 文案必须同时具备节气感、画面感、产品关联和品牌气质。
4. 海报主文案要短，适合直接放在画面中；备选文案可以略有层次，但不能写成长段抒情。
5. 节气意象不能只停留在节气名称，要体现该节气的自然变化、物候、温度、光线、风、雨、作物或生活感。
6. 产品卖点要融入画面和情绪，不要生硬堆参数。
7. 不编造市场规模、销售数据、功效承诺、认证背书、专家推荐、客户案例或用户评价。
8. 食品、宠物食品、保健、美妆、母婴、医疗健康等品类，避免夸大功效和治疗暗示。
9. 生图提示词必须具体到主体、场景、构图、色彩、光线、材质、产品呈现、文字排版和负面提示词。
10. 如果可用工具支持直接生图，应在输出提示词后调用生图能力生成成品海报；如果当前环境不支持生图，则清楚说明只能交付提示词。

## 工作流程

### 第一步：理解输入

快速整理用户信息：

| 项目 | 内容 |
|---|---|
| 节气 | ... |
| 产品 / 品类 | ... |
| 核心卖点 | ... |
| 品牌调性 | ... |
| 目标人群 | ... |
| 海报用途 | ... |
| 关键假设 | ... |

### 第二步：提炼节气与产品共鸣点

从节气自然特征中找到能和产品价值相接的点。

优先使用以下方向：

- 气候变化：冷暖、湿润、暑气、秋燥、寒意。
- 物候变化：草木、花果、虫鸣、雨雪、风露。
- 身体感受：轻盈、舒展、蓄养、清爽、安定。
- 生活场景：窗边、餐桌、晨光、归家、夜晚、出行、换季。
- 产品价值：口感、香气、营养、舒适、效率、陪伴、仪式感。

输出 2-3 句话说明节气与产品如何结合。

### 第三步：生成海报文案

输出：

- 1 句主文案：最适合放在海报上的一句。
- 5 句备选文案：从不同角度提供选择。

文案要求：

- 主文案尽量控制在 20 字以内，必要时可略长。
- 每句都要有具体意象或情绪动作。
- 不要把产品名称机械塞进每一句。
- 不要连续使用同一种句式。
- 至少 3 句能直接放进海报。

### 第四步：构建视觉创意

说明海报画面：

- 画面主体
- 产品呈现方式
- 节气自然意象
- 场景空间
- 构图关系
- 色彩系统
- 光线和氛围
- 字体与排版方向

视觉创意要能直接指导设计或生图，不写抽象形容词堆砌。

### 第五步：生成 AI 生图提示词

提示词必须包含：

- 海报比例和用途
- 主体与产品
- 节气场景和自然意象
- 构图与留白
- 色彩和光线
- 材质和质感
- 中文文字内容
- 字体和排版风格
- 商业海报完成度要求
- 负面提示词

如果模型不擅长生成准确中文文字，应在提示词中保留文字区，并提醒可后期排版：

```text
顶部保留干净文字区，可后期加入中文标题；如果生成文字不准确，请保持文字区留白。
```

### 第六步：生成成品海报

如果当前环境有图像生成能力，使用最终提示词直接生成一张成品海报。

生成前不要再追问，除非用户明确要求确认风格。

如果生成结果中文字不准确，应在回复中说明：

> 成品图中的中文可能需要后期排版校正，核心画面和版式可作为海报底图使用。

默认按照以下结构输出：

## 视觉创意

## 质量标准

合格输出应满足：

- 节气特征明确，不是把节气名贴在普通产品图上。
- 产品价值自然进入画面，不像硬广堆卖点。
- 主文案有记忆点，且能和画面互相解释。
- 备选文案之间角度不同，而不是同一句换词。
- 生图提示词足够具体，设计师或 AI 模型都能执行。
- 没有编造数据、功效、背书或真实品牌资产。
- 成品海报具备商业传播质感，适合继续精修使用。

## 示例

用户输入：

```text
立夏，宠物猫粮
```

可输出方向：

- 节气洞察：立夏是热意初生、万物舒展的时刻，猫粮表达“轻盈营养”和“唤醒食欲”。
- 主文案：立夏一到，风把猫的胃口吹亮了
- 画面：清晨窗边、浅色家猫、陶瓷猫碗、初夏新绿、青梅、薄纱帘、柔和自然光。
- 提示词：竖版 4:5 高端宠物食品节气海报，猫和产品位于视觉中心偏下，顶部留出中文标题区，米白、嫩绿、浅金、原木色，真实摄影质感。
